# Dashboard Appliance Image

## Provided services

* Mosquitto MQTT
* Node Red
* Avahi
* Feinstaub Dashboard

## How to start

* Clone this repo and enter the folder appliance
* `sudo git clone`
* `sudo chown -R 1000:1000 ./nodered`
* `sudo docker-compose up -d`

* Go to your-host-ip:1880/ui
