# Dashboard Appliance Image

## Provided services

* Node Red + Custom Dashboard for Air Quality Monitoring


## How to start

* `sudo docker run -it -p 1880:1880 digitalhhz/nodered-dhhz`
* Go to your-host-ip:1880/ui
